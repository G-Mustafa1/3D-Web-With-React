## ğŸ”¹ Project Overview

**My3DStudio** is a **modern 3D web studio website** built using **React, Three.js, Tailwind CSS, and Framer Motion**.  
It showcases **interactive 3D experiences** for brands, products, and services with performance-first design and smooth animations.

**Pages Included:**

- **Home Page:** Hero section with interactive 3D canvas, call-to-action buttons.  
- **About Page:** Company information with 3D visuals.  
- **Services Page:** Highlights immersive 3D services and solutions.  
- **Contact Page:** Hero section with animated contact form and scroll-to-contact feature.
